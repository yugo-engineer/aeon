## What is Aeon?

Aeon is a background AI agent that lives in your GitHub repo. It runs skills — small markdown instruction files — on a cron schedule. Each skill is a focused task: write an article, fetch the day's top HN stories, watch a token price, generate a changelog.

No server. No daemon. If GitHub Actions is up, Aeon is up.

**50 skills** across four categories:

| Category | Skills | What it does |
|----------|--------|-------------|
| Research & Content | `article`, `digest`, `rss-digest`, `paper-pick`, … | Write articles, summarize feeds, surface papers |
| Dev & Code | `changelog`, `pr-review`, `code-health`, `feature`, … | Monitor repos, review PRs, build features |
| Crypto & Markets | `token-alert`, `polymarket`, `defi-overview`, … | Track prices, DeFi positions, prediction markets |
| Meta / Agent | `heartbeat`, `memory-flush`, `self-review`, … | Keep Aeon healthy and self-improving |

## Quick Start

```bash
git clone https://github.com/aaronjmars/aeon
cd aeon && ./aeon
```

Open `http://localhost:5555`, add your Claude API key or OAuth token, pick skills, and push. GitHub Actions handles the rest.
